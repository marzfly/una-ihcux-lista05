## ⋆˚✿˖° Heurísticas - resumo  

**ᯓ★ 1. Visibilidade do status do sistema**  
Ao usuário são apresentadas informações sobre o status do sistema, como [CARREGANDO...] e [Calculando taxas]. Isso ajuda o user a estar ciente do que está acontecendo e não ter que se preocupar desnecessariamente.  

**ᯓ★ 5. Prevenção de erros**  
O código que nós usamos foi projetado justamente para evitar erros do usuário. Isso porque ele é capaz de executar blocos de código específicos e está preparado para tratar erros que possam vir a ocorrer.

**ᯓ★ 8. Estética e design minimalista**  
Essa heurística preza por um design sem informações excessivas. Esteticamente agradável e coeso, mas sem informações demais.
